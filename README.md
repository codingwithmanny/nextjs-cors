# NextJS13 AppRouter Cors Configuration

A simple example of how to configure CORS in a NextJS 13 app using AppRouter.

---

## Getting Started

### Prerequisites

- NVM or [NodeJS v18.17.1](https://nodejs.org/en/download/)
- pnpm

### Installation

```bash
#FROM: /

pnpm install
```

### NextJS Server

```bash
#FROM: /

pnpm dev
```

### Client-Side App Server

```bash
#FROM: /

pnpm client
```
